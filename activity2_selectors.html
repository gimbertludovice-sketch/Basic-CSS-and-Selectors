Activity 2 – Class and ID Selectors

CSS Selectors in Action

An ID selector (e.g. #title) targets a single unique element on the page. It is the most specific selector, meaning it overrides class and element rules when there is a conflict. Use it for elements that appear exactly once.

A class selector (e.g. .info) can be applied to many elements at once, making it perfect for reusable styles like this beige background, 16px font size, and comfortable padding applied to both paragraphs here.
