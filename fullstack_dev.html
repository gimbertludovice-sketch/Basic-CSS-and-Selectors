Full Stack Development

Full stack development covers both the client-facing front end and the server-side back end of a web application — plus the databases and infrastructure that tie them together. A full stack developer can build and ship a complete product independently.

This page demonstrates real full stack concepts while applying the same CSS rules you learned in Activity 2: #id selectors for unique green centered headings, and .class selectors for reusable beige info blocks with 16px text and padding.

Front End

The front end is everything a user sees and interacts with in the browser. It is built with HTML for structure, CSS for styling, and JavaScript for interactivity. Modern frameworks like React, Vue, and Angular make it easier to build complex, dynamic user interfaces.

HTML5 CSS3 JavaScript React Vue

Responsive design ensures pages look great on any screen size. Tools like Flexbox, CSS Grid, and media queries let developers adapt layouts for mobile, tablet, and desktop without duplicating code.

Back End

The back end runs on a server and handles business logic, authentication, and communication with databases. Common languages include Node.js, Python, PHP, and Java. Frameworks like Express, Django, and Laravel provide structure and speed up development.

Node.js Python Express Django REST API

APIs (Application Programming Interfaces) connect the front end to the back end. A REST API sends and receives JSON data over HTTP, letting the browser request only the data it needs without reloading the whole page.

Database

Databases store all persistent data — user accounts, posts, orders, and more. Relational databases like MySQL and PostgreSQL organize data in tables with rows and columns. NoSQL databases like MongoDB store flexible, document-based data that can change shape over time.

MySQL PostgreSQL MongoDB Firebase

SQL (Structured Query Language) is the standard way to interact with relational databases. Commands like SELECT, INSERT, UPDATE, and DELETE let developers read and write data with precision.

DevOps & Deployment

Once an application is built, it needs to be deployed so real users can access it. Platforms like Vercel, Netlify, and Railway make deploying front and back end apps simple. Docker packages an app and all its dependencies into a container so it runs the same anywhere.

Docker Vercel Netlify GitHub Actions CI/CD

Version control with Git and GitHub lets teams collaborate safely. CI/CD pipelines automatically test and deploy code every time a developer pushes a change, reducing human error and speeding up releases.
